## Step 1: Project Setup

First, we started with TinaCMS's Next.js starter template using `pnpm`:

```bash
# Create a new TinaCMS project
pnpm dlx create-tina-app@latest hark-blog --template tina-cloud-starter

# Navigate to the project
cd hark-blog

# Install dependencies
pnpm install

# Start the dev server
pnpm dev
```

**Pro Tip**: TinaCMS requires an empty directory, so move any existing files temporarily during setup.

The starter gives you:
- üì¶ **Next.js 15** with Turbopack for blazing-fast compilation
- ü¶ô **TinaCMS** with file-based content management
- üé® **Tailwind CSS v4** for styling
- üìä **GraphQL API** for content queries

## Step 2: Modern Tech Blog Aesthetic

This is where GitHub Copilot really shined! We transformed the default blog into a modern masterpiece.

### Color System Overhaul

We rebuilt the entire color system using `oklch()` color space for better consistency across light and dark modes:

```css
:root {
  /* Light mode - Clean white with red accent */
  --primary: oklch(0.55 0.22 25);
  --secondary: oklch(0.95 0.05 60);
  --background: oklch(0.99 0 0);
  --foreground: oklch(0.15 0 0);
}

.dark {
  /* Dark mode - Deep charcoal with vibrant accents */
  --primary: oklch(0.65 0.25 25);
  --secondary: oklch(0.25 0.03 60);
  --background: oklch(0.12 0 0);
  --foreground: oklch(0.95 0 0);
}
```

**Why oklch()?** It provides perceptually uniform colors that look consistent across different lightness levels - perfect for dark mode!

### Custom Utility Classes

We added custom CSS utilities for that modern tech blog feel:

```css
/* Gradient text effect */
.gradient-text {
  @apply bg-gradient-to-r from-primary via-purple-500 to-pink-500 
         bg-clip-text text-transparent;
}

/* Frosted glass effect */
.glass {
  @apply backdrop-blur-md bg-background/50 border border-border/50;
}

/* Card hover animation */
.card-hover {
  @apply transition-all duration-300 hover:shadow-lg 
         hover:shadow-primary/10 hover:-translate-y-1;
}
```
